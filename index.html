HTTP Client

{% highlight clojure %} (let [resp1 (http/get "http://http-kit.org/");; request concurrently, return a promise resp2 (http/get "http://clojure.org")] ;; handle one by one, wait as necessary (println "resp1's status: " (:status @resp1)) (println "resp2's status: " (:status @resp2))) {% endhighlight %}
